# Before — Trip Sage (a ChatGPT Custom GPT)

This is the source material exactly as it lives inside ChatGPT. None of it is a
file Maya can open, version, or move to another tool.

Files here are *representations* of what is locked inside the GPT:

- [`custom-gpt-instructions.md`](custom-gpt-instructions.md) — the GPT's system
  instructions (Configure → Instructions).
- [`saved-memory.md`](saved-memory.md) — the GPT's saved-memory lines
  (Settings → Personalization → Memory).
- [`knowledge/`](knowledge/) — the two uploaded knowledge files:
  - `packing-checklist.txt`
  - `tokyo-3-day-itinerary.txt`
- [`conversation-export.md`](conversation-export.md) — one exported chat that
  contains a decided lesson worth keeping.

In ChatGPT these are spread across four different screens and none of them links
to the others. After migration they become one folder where the project file
links to everything.
